Working safely

You are always in control.

Set the right permissions, review output before it ships, and keep sensitive data out of the folder.
Before anything ships
✓ Read what Cowork changed
✓ Open the output file and check it
▶ Approve - or redirect and re-run

Permission modes

You choose what Cowork does without asking.

Start cautious. Loosen permissions only once you trust the workflow.
Permission options
  • 1Ask every time - safest, recommended to start
  • 2Allow within this folder - unlocks for trusted workflows
Settings › Permissions. Start on option 1 and move to 2 only when you are confident in the skill.
Sensitive data

Only open folders you are comfortable sharing.

Grant the minimum a connector asks for. If a folder has customer data or credentials, keep it out of the Cowork workspace.
Folder access prompt
Allow Claude to read and edit files in verdal-marketing?

This folder contains brand context and templates only - no customer data.

Check the output

Cowork can sound confident and be wrong.

Before you publish anything, trace every fact, stat, or quote back to the file it should have come from.
Ask this before you publish
Which file in my folder says plant food sales are up 40 percent?
The rule
If it is not in your files, Cowork made it up. Never publish a claim you did not give it the source for.
When not to use AI

Some things stay human.

A short list of tasks to keep off Cowork's plate entirely.
Keep these off Cowork
  • 1A real, verified human quote or attribution
  • 2Anything needing legal or compliance sign-off
  • 3Facts where a wrong answer has real cost - pricing, medical-style claims, competitor facts you will publish
  • 4Anything involving real customer personal data
