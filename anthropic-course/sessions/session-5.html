Live artifacts

A live artifact is a dashboard that refreshes.

Build it once. Pin it. Click refresh for new data - no re-prompting needed.

How to build one
  • 1Turn on Artifacts and Code Execution in Cowork settings
  • 2Use the words "live artifact" in your prompt
  • 3Pin it to your sidebar - click refresh anytime for new data
Tip - the words that matter
Say "live artifact" in the prompt or Cowork will produce a static file instead.
"Build a live artifact for Verdal showing..."
vs.
"Build a dashboard for Verdal..." ← static
Settings path: Cowork preferences → Features → Artifacts + Code Execution → On

Big jobs, done in parallel

Split the work, track the list.

For a large batch, tell Cowork to run parts in parallel and keep its own checklist. Then spot-check that nothing was dropped.

Generate a carousel for all 12 topics in data/content-planner.csv. Run them in parallel, keep a checklist of done vs pending, and give me the checklist at the end so I can spot-check.
