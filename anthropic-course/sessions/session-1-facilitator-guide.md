# Session 1 facilitator guide - Meet Claude Cowork

**Length:** 2 hours. **Audience:** non-technical marketers. **Example:** Verdal. **You need:** the Verdal folder on the demo machine, Cowork installed and signed in, this guide, and the slides.

Goal for the room: by the end, every attendee has handed Cowork a real task on their own laptop and watched it produce a finished file. They leave believing Cowork is different from Chat.

Capabilities covered: the task loop, setup and permissions, working with files, the model picker.

## 0:00 - 0:05  Recap and promise

Say: "Most people install Cowork and keep using it like the chat box. Today you will see why it is a different tool. By the end of this session you will have handed Cowork a real task and watched it produce a finished file on your own machine."

Set the one rule: watch the demo first, then you do it. Nobody types until the lab.

## 0:05 - 0:30  Teach - Chat vs Cowork and the task loop

Keep this plain and short. Cover, in order:

1. **Chat vs Cowork.** Chat runs in the cloud and hands you text you copy out. Cowork is a local desktop agent: you give it a folder, it reads your files, does multi-step work, and saves finished output back into the folder. For most marketing work, Cowork is where the real work happens.
2. **The task loop.** When you give Cowork a task, it makes a plan, works through it step by step, and shows a to-do list as it goes. You can watch and steer. It is not one big black box.
3. **Setup and permissions.** Cowork only touches the folder you point it at, and asks permission the first time. You are always in control of what it can see.
4. **Files are the point.** Cowork reads and writes real files: docs, decks, sheets, images. The output lands in your folder, not in a chat window.
5. **The model picker.** You can pick which model runs a task: a heavier one for hard thinking, a faster one for simple jobs. Show where the picker sits.

Analogy that lands: "Chat is a smart colleague on a phone call. Cowork is that colleague sitting at your computer, with your files open, doing the work."

## 0:30 - 0:55  Demo - strategy deck from Verdal

This is the headline "wow". Follow the demo script exactly (see `session-1-demo-script.md`, to be built in Phase 3). High level:

1. Open Cowork. Load the Verdal marketing folder. Point out the permission prompt and grant it.
2. Show the `context/` files quickly so the room sees Cowork has real brand knowledge to work from.
3. Paste the demo prompt: ask Cowork to read the brand context and build a 12-slide on-brand marketing strategy deck, saving it into the projects folder.
4. Narrate the task loop out loud as it runs: "See it reading the brand context first. Now it is building the outline. Now it is generating slides."
5. Open the finished deck. Point out it used the real Verdal positioning and colors, not generic filler, and it saved into the folder.

Say while it runs: "I did not paste any brand details. It knew them because they live in the folder. That is the whole idea."

If the demo is slow, keep talking through the task loop; do not sit in silence. If it fails, switch to the recorded backup clip (Phase 5 deliverable).

## 0:55 - 1:00  Break

## 1:00 - 1:45  Lab - your first task

Attendees work on their own laptops with their own Verdal copy. Walk the room.

The lab worksheet (Phase 3 deliverable) has the exact steps. High level:
1. Open Cowork, load your Verdal folder, grant access.
2. Hand Cowork a first task, for example: "Read the brand context and write three on-brand Instagram captions about repotting, saved to the projects folder."
3. Watch the task loop. Open the file it saved.
4. Try the model picker: rerun with a different model and notice the difference.

Login note: if some attendees are on a common ID, they follow along while the volunteer drives, and do the real task as homework. Confirm this before the session.

Circulate. The two most common blockers: folder access not granted, and looking for output in the chat instead of the folder. The FAQ sheet (Phase 4) has fixes.

## 1:45 - 1:55  Debrief and Q&A

Ask three questions: What surprised you? Where did it save the file? How was this different from Chat? Let a few people answer. This cements the mental model.

## 1:55 - 2:00  Preview and homework

Preview Session 2: "Next time we make Cowork yours, so it remembers your brand and your rules without you repeating them."

Homework (5 minutes): hand Cowork one more task on Verdal, any task you like, and see what it makes.
