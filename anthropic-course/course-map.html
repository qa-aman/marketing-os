Claude Cowork workshop

Course map

Six 2-hour sessions, grouped into three phases by difficulty. All hands-on in the Claude Cowork desktop app. No terminal, no code.

Scope in one line. This workshop teaches Claude Cowork, and pairs it with browser tools like Perplexity and Gemini where useful. The only thing left out is Claude Code (terminal work), because it is too technical for this group. See the coverage map for exactly what is in and what is out.
