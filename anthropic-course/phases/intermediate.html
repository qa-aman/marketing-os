Build & connect

Turn repeat work into reusable skills, and connect Cowork to the tools you already use every day.

Who this is for. Anyone who has finished Phase 1 and can run a task in Cowork and set up a Project. You do not need any coding skill here either, connectors are a few clicks.

Session 3 - Skills & plugins

What a skill is · when to build one · the official marketing plugin

A skill packages a workflow so Cowork repeats it on demand with the same quality. You learn a simple rule for when something is worth turning into a skill, two ways to build one without starting from a blank page, and how to install ready-made skills.

  • Skills vs Projects vs plugins, in plain language
  • The "should this be a skill?" rule: repeated 3+ times, quality has to be consistent
  • Build a Verdal weekly-brief skill, then run it
  • Install the official Anthropic marketing plugin for brand review, campaign plan, and competitive brief
You can now: build and run a skill, and install ready-made ones so you leave with working tools.

Session 4 - Connect your tools

Connectors · the safety order · Claude in Chrome · Notion · Apify

Connectors let Cowork read live data and work with your real tools, not just local files. You learn the one rule that keeps this safe, and connect the tools a marketer actually uses.

  • What a connector is, and the safety order: native > official > community, grant the minimum
  • Claude in Chrome: let Cowork read real web pages for you
  • Notion: push a content calendar and build a dashboard
  • Apify: scrape competitor social posts into a spreadsheet and dashboard
You can now: connect a tool you already use and pull something real from it into a Verdal deliverable.
