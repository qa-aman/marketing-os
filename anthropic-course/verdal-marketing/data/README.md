# data

Realistic but made-up numbers so the labs work without connecting a live tool.

| File | Used in | What it is |
|---|---|---|
| `keyword-opportunities.csv` | Session 5 | SEO keywords, search volume, difficulty, whether Verdal ranks |
| `competitor-posts.csv` | Session 5 | Sample social posts from Verdal and three competitors, with engagement |
| `last-month-metrics.csv` | bonus | Channel performance last month: followers, posts, engagement, orders, revenue |
| `content-planner.csv` | bonus | A four-week content calendar: topic, format, persona, channel, status |

## Bonus prompts (self-directed practice)

The last two files are not used by a guide, but they are great for extra practice. Try these in Cowork:

- "Read data/last-month-metrics.csv and tell me which channel to double down on next month, with one reason. Save it to projects/channel-focus.md"
- "Read data/content-planner.csv and context/, then draft the first piece on the calendar in Verdal's voice. Save it to projects/."

When you rebuild these exercises on your own brand, swap in your real exports the same way.
