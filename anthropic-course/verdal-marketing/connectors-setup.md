# Setup and connectors - for this Cowork course

Everything you need to install and connect for the six sessions, with official links. This course is Cowork only. No terminal, no code. You only need a tool when the session that uses it comes up, so set things up as you go.

If a step is unclear, come here first, then follow the official link.

## Before Session 1 - install Cowork

- Download Claude Desktop, sign in, and confirm Cowork is in the sidebar.
- Get started guide (official): https://support.claude.com/en/articles/13345190-get-started-with-claude-cowork
- Desktop app help: https://code.claude.com/docs/en/desktop
- You need a paid Claude subscription (Pro, Max, or Team). The free tier will not run these workflows.
- Account or billing issues: https://support.anthropic.com

## Where connectors live

In Claude Desktop, connectors are under **Customize > Connectors**. You will see built-in options (Google Workspace, Microsoft 365, and more) plus a way to add others. Connect a tool once and Cowork can use it in any session.

General tips:
- Grant the minimum access a tool asks for. Read-only is usually enough.
- Test a connector with one small request before a big one.
- If a connector is greyed out or failing, disconnect and reconnect, then retry.

## Session 4 - connect your tools

**Google Workspace** (Drive, Gmail, Calendar).
- Customize > Connectors > Google Workspace > sign in with Google and allow read access.

**Microsoft 365** (Outlook, OneDrive, SharePoint, Teams).
- Customize > Connectors > Microsoft 365 > sign in with your work account and allow access.

**Claude in Chrome** - lets Cowork browse real websites.
- Enable Claude in Chrome under Customize > Connectors.
- Official setup docs: https://code.claude.com/docs/en/chrome
- Background: https://www.anthropic.com/news/claude-for-chrome
- Install the official Claude extension from the Chrome Web Store: https://chromewebstore.google.com (publisher: Anthropic), then pin it. Install only the official one.

**Other connectors** (optional, mention-only in the session).
- Under Customize > Connectors you will also find tools for things like image generation (ad visuals and header images), web scraping, and deck building. Add one the same way if a task needs it. For the labs in this course you do not need them, the kit already has the data.

## Session 5 - turn on live artifacts

Live artifacts need two capabilities switched on first.
- Go to Settings (or Preferences) > Capabilities and turn on **Artifacts** and **Code Execution**.
- Scheduled tasks are built in, no extra setup. You create one from the Schedule area of a project.

> The exact menu labels and button locations can move between Cowork releases. Verify the paths above the morning of the session.

## What you do NOT need for this course

No terminal, no Claude Code, no coding. If a guide or search result points you at install scripts, MCP servers, or command-line steps, that is a different, more technical course. Everything here happens by clicking in the Claude desktop app.
