# Carousel design system - a Cowork project

This folder is a ready-made project for Session 2. In Cowork, create a project from this folder. It has everything Cowork needs to turn any piece of content into an on-brand Verdal carousel, the same way every time.

## What's inside

```
carousel-design-system/
  context/            the Verdal brand context (voice, brand, ICP)
  reference/
    carousel-examples.md   the layouts Verdal likes
  output/             finished carousels save here
```

## How to use it in Session 2

1. In Cowork, go to Projects, create a new project, and choose "use an existing folder". Pick this folder.
2. Ask Cowork to read the reference examples and brand context, then write a brand style guide and a carousel design system file, saved in the project.
3. Set the project instructions so every future carousel follows the same rules.
4. Then hand it a piece of content and ask for a 7-slide carousel. It saves to `output/`.

Because this is a project with its own context and memory, you set it up once and reuse it. That is the whole point of the projects lesson.
