# LinkedIn Post - marketing-os

---

1000+ hours in Claude Code.

Somewhere in that time I stopped asking "what can I build for myself" and started asking "what can I build that helps entire teams."

Marketing was first.

I built marketing-os - a free, cloneable OS for marketing teams built on Claude Code. The goal: make your marketing team 10x faster without adding headcount.

Here's what's inside:

**21 skills** covering every job a marketing team does weekly:
- LinkedIn posts, thought leadership, newsletters, SEO articles
- Campaign briefs, webinar plans, content calendars
- A/B copy variants, email sequences, landing page copy
- KPI reviews, campaign retros, competitor analysis
- Case studies, press releases, pitch decks

**A knowledge layer** that every skill reads from - your brand voice, ICP, positioning, past learnings. Output sounds like your brand, not generic AI.

**`/onboard`** - the setup wizard. Drop your existing brand docs into the repo (pitch deck, brand guidelines, past articles), run the ingest script, and the wizard scores your context against an internal rubric. Won't hand you to the skills until you're above 95/100.

**A learning loop** - every campaign retro writes to a shared learnings file. Future campaigns start smarter.

**How to start:**
1. Clone the repo (link in first comment)
2. Install: `pip install pypdf python-docx python-pptx`
3. Open in Claude Code
4. Run `/onboard`
5. Start shipping

Two vertical examples included so you can see what good looks like before you fill in your own context - one for B2B SaaS, one for a consulting firm.

This is v0.1. I'm building more team OSes next.

Feedback genuinely appreciated - what's missing, what broke, what you'd add.
