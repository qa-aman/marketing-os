# Module 3 - Claude Skills Deep Dive

**Track:** power · **Level:** intermediate · **Builds on:** Modules 1-2

Power track. This module uses Claude Code (the desktop app and sometimes the terminal) alongside Cowork. You do not need to be a developer, but you will run slash commands, edit a `CLAUDE.md` file, and package files into a plugin. Every step is explained plainly.

You move from using skills to designing, chaining, and shipping your own skill library.
