# Verdal - worked example brand

This is the fictional brand used through the whole workshop. Follow along with Verdal first, then redo each exercise with your own brand.

Copy this folder onto your computer to use it as your Cowork workspace, or just read the files to see what "good context" looks like.

## What's in here

```
verdal/
  context/
    brand-context.md            who Verdal is, what it sells, positioning
    brand-voice.md              how Verdal sounds, words to use and avoid
    ideal-customer-profile.md   who we sell to
  templates/
    strategy-deck-outline.md    reusable 12-slide deck structure (Lesson 2)
    carousel-design-system.md   carousel layout rules (Lesson 3)
  projects/                     where Cowork saves finished work
```

## The brand in one paragraph

Verdal makes premium indoor plant care products (plant nutrients, precision tools, and a monthly care box) for people who keep a lot of houseplants and treat them like pets. Direct-to-consumer through its own site and a few premium plant shops. Brand color is a deep botanical green. Friendly, expert, never intimidating.

## How to use it

- **Lesson 1:** point Cowork at this folder so it reads the `context/` files.
- **Lessons 2-8:** every exercise uses these files as the brand truth.

When you redo the exercises for your own brand, you will create your own `context/` folder the same way.
