# Module 5 - Your AI Marketing Team

**Track:** power · **Level:** advanced · **Builds on:** Modules 3-4

Power track and the capstone. This module uses Claude Code, the `/agents` command, and routing rules in `CLAUDE.md` to build a team of specialized agents with a lead that delegates. The most technical module in the course, explained step by step.

You end with a multi-agent team that runs a full campaign from one prompt, with a Notion board as the shared human-AI task queue.
