# knowledge.example/saas/

A complete, fictional example of a populated `knowledge/` folder for a B2B SaaS company. Use this as a reference for what depth and specificity to aim for when filling in your own `knowledge/`.

**Fictional company**: Threadline. A customer onboarding automation platform for B2B SaaS. Series B, 80 employees, founded 2022, HQ Austin.

## How to use this folder

1. Read each file to see the level of detail expected.
2. Compare against your own `knowledge/`. Where are you vague? Fix it.
3. Do NOT copy these files into `knowledge/`. They are fictional and will pollute your output.

## Why this exists

Marketing skills only produce excellent output when they have excellent context. The single biggest failure mode of a Marketing OS is half-filled `knowledge/`: skills produce generic AI slop because they have no specific brand, voice, or persona to anchor to.

This example sets the bar.
