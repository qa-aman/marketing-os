# knowledge.example/services/

This is a complete fictional example for a **B2B professional services firm** - specifically a go-to-market consultancy called Clearpath Advisory.

Use it as a reference for how to populate `knowledge/` when your company is:
- A consultancy, agency, or advisory firm
- A professional services business
- A company where the "product" is expertise and execution, not software

## How it differs from the SaaS example

| | `knowledge.example/saas/` (Threadline) | `knowledge.example/services/` (Clearpath) |
|---|---|---|
| Revenue model | ARR, subscription tiers | Retainer + fixed project fees |
| ICP trigger | Activation rate stuck, no-engineering onboarding | Founder-dependent pipeline, first marketing hire |
| KPIs | Trial-to-paid, DAU, activation rate | Pipeline generated, inbound %, client NPS |
| Content strategy | PLG audience (growth leads, PMs) | Buying committee (VP Marketing, CEO) |
| Sales cycle | 21-35 days, self-serve option | 30-60 days, no self-serve |
| Voice | Operator-to-operator, fast, specific | Advisor-to-peer, plainspoken, earned confidence |

## Files in this example

```
services/
├── company.md              # Clearpath Advisory: who they are, what they do, who they serve
├── role.md                 # Director of Marketing: KPIs, working style, what they want from the OS
├── kpis.md                 # Pipeline, inbound %, newsletter, organic search targets
├── learnings.md            # Q1 content retro + webinar retro with reusable insights
├── brand/
│   ├── voice.md            # Advisor voice: specific, earned confidence, short sentences
│   └── visual.md           # Navy + gold palette, Freight Display + Inter
├── services/
│   └── demand-gen-build.md # Core service: 6-12 month embedded engagement, pricing, proof points
├── icp/
│   └── personas.md         # Priya (VP Marketing, Series B) + Marcus (CEO)
├── markets/
│   ├── positioning.md      # Embedded GTM consultancy, 90-day pipeline, we leave behind a machine
│   └── competitors.md      # Demand gen agencies, fractional CMO firms, in-house hire
└── content-library/
    ├── why-your-icp-is-a-job-title.md    # Blog post in Clearpath voice
    └── case-studies/
        └── stytch-pipeline-in-8-months.md  # Case study: $2M pipeline in 8 months
```

## Using this example

1. Read `brand/voice.md` to understand how a consulting firm sounds different from a SaaS product
2. Read `icp/personas.md` to see how situation-based ICP (not just job title) is defined
3. Read `services/demand-gen-build.md` to see how a retainer service is documented with pricing and objections
4. Read `content-library/why-your-icp-is-a-job-title.md` to see consulting firm voice in practice

When populating your own `knowledge/`, replace all Clearpath/fictional content with your actual company details.
